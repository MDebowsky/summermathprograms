Executive Director Search

The Summer Mathematics Programs Consortium is excited to announce our search for our first Executive Director.

About the Search

The full position description can be found here.

We will review applications on a rolling basis until the position is filled. We are flexible on the exact start date but prefer as soon as possible. Ideally, the selected candidate will be able to participate in our annual conference (November 14–16, 2025) even if this is before the official start date.

Apply

We are accepting applications online: click here to apply. Note that instead of a cover letter, we ask candidates to respond to a set of short-answer questions. We also ask you to upload a resume/CV.

If you have any questions, please email the Steering Committee.
